# Simple U-boat TFC (Torpedo Fire Control)

**Disclaimer:** This is a personal project developed by **bidope**.  
It is not an official military or commercial software.

---

### 프로젝트 소개 (Overview)
이 애플리케이션은 어뢰 데이터 컴퓨터(TDC)에 대하여 현대적인 프로그램 툴을 사용한 개인용 사격통제장치입니다.
취미로 시작한 프로젝트이지만, 실제 유보트 시뮬레이션 게임에서 정밀한 어뢰 공격을 할 수 있도록 돕는 실용적인 계산 장치입니다.

---

### 🇰🇷 Language Support
* **Interface:** Fully localized in Korean (한국어 지원).
* 이 앱은 유보트 시뮬레이션을 즐기는 한국 사용자들을 위해 최적화되었습니다.

---

### Tech Stack
* **Language:** Python
* **Library:** Flet (Flutter UI Framework)
* **Tools:** Git, Linux(Ubuntu)
  
---

### 만든 동기 
심심해서 어뢰 TDC 계산 프로그램 제작, 추후 게임에 사용할것임.
