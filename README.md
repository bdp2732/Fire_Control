# Simple U-boat TFC (Torpedo Fire Control)

**Disclaimer:** This is a personal project developed by **fermion2732**. 
It is not an official military or commercial software.

### Overview
This application is a simplified version of the historical Torpedo Data Computer (TDC), 
designed for mobile devices using Python and Kivy.

---

### 🇰🇷 Language Support
* **Interface:** Fully localized in **Korean** (한국어 지원).
* This app is specifically designed for Korean-speaking U-boat simulation enthusiasts.
